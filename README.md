# 🎬 Jjojo Otaku

Modern, professional anime streaming platform built with Node.js and Express.

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![Node](https://img.shields.io/badge/node-%3E%3D18.0.0-brightgreen.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## ✨ Features

- 🔐 **User Management** - Registration & login system with secure password hashing
- 📺 **Complete Anime Streaming** - Full integration with Otakudesu API
- 🏠 **Home Page** - Featured anime, ongoing & completed sections
- 📅 **Release Schedule** - Weekly anime schedule
- 🔍 **Search** - Real-time anime search
- 🎭 **Genres** - Browse anime by genre
- 📱 **Responsive Design** - Mobile-first, works on all devices
- 🌙 **Dark Theme** - Professional dark UI design
- 💾 **Watch History** - Track watched episodes (logged in users)
- ⏱️ **Progress Tracking** - Resume watching from where you left off
- ☁️ **GitHub Sync** - Data synchronization with GitHub repository

## 🚀 Tech Stack

- **Backend**: Node.js, Express.js
- **Template Engine**: EJS
- **Styling**: Custom CSS (no frameworks)
- **Authentication**: bcryptjs, express-session
- **HTTP Client**: Axios
- **Data Storage**: JSON files (GitHub sync)
